== L'API Shortcode ==

Introduite dans WordPress 2.5 l'API Shortcode est un simple ensemble de fonctions qui permettent de créer des macro-codes pour une utilisation dans le contenu des articles. Par exemple, le shortcode suivant (dans le contenu de l'article/page) ajoutera une galerie photo dans la page :
<pre>[gallery]</pre>
